雲端同步設定

已內建你的 Supabase 設定，開檔即自動同步。其他裝置打開同一個網址也會自動連到同一份白板。

備援：若要自行重建資料表，在 Supabase 的 SQL Editor 貼上並 Run： 
create table if not exists whiteboard (
  id text primary key,
  data jsonb,
  updated_at bigint,
  by text
);
alter table whiteboard enable row level security;
create policy "public rw" on whiteboard
  for all using (true) with check (true);
